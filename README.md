🗺️ Crime Incidents in Washington, D.C. — GIS Analysis Project
📌 Project Summary

This project leverages Geographic Information Systems (GIS) to analyze crime incidents in Washington, D.C. (2024), uncovering spatial and temporal patterns that can support data-driven public safety decisions.

Through geospatial analysis and mapping, I identified crime hotspots, high-risk zones, and time-based trends, demonstrating how GIS can translate raw data into actionable insights.

📊 Key Insights
🔥 Crime Hotspots
High crime density concentrated in central and densely populated areas
Clear geographic clustering rather than uniform distribution
🏙️ Ward-Level Analysis
Wards 1, 2, and 5 have the highest crime rates
Ward 1 identified as a critical focus area
🧾 Crime Type Patterns
Theft-related incidents dominate across all wards
Other crimes include robbery, burglary, and vehicle theft
⏰ Time-Based Trends
Evening hours show the highest crime activity
Followed by daytime, with midnight being the lowest
